# Plan: Enforce Issue Tagging in PRs for Auto-Close on Merge

**Issue**: [#49](https://github.com/The-AgenticFlow/AgentFlow/issues/49)

## Summary

Implement a pre-merge validation in VESSEL that checks if PR bodies contain a valid closing keyword (`Closes|Fixes|Resolves #\d+`) referencing an issue. If no reference is found, block the merge and route to NEXUS for decision-making.

1. **FORGE PR creation** (`crates/agent-forge/src/lib.rs:837-846`): Already includes `Resolves #XXX` in PR body
2. **PR fetching** (`crates/github/src/rest.rs:425-451`): `get_pull_request` returns `PrInfo` without body field, but internally uses `resp.body` for ticket extraction
3. **Vessel merge flow** (`crates/agent-vessel/src/node.rs`): Processes PRs through `process_single_pr` → CI poll → merge
4. **Existing patterns**: 
   - `VesselOutcome` enum for various outcomes (merged, ci_failed, conflicts, etc.)
   - `VesselNotifier` for emitting events
   - Action constants for routing (`ACTION_CI_FIX_NEEDED`, `ACTION_CONFLICTS_DETECTED`, etc.)

### 2. Add closing keyword validation function

**File**: `crates/agent-vessel/src/validation.rs` (new file)

```rust
use regex::Regex;

/// Valid closing keywords per GitHub documentation
const CLOSING_KEYWORDS_PATTERN: &str = r"(?i)(close[sd]?|fix(e[sd])?|resolve[sd]?)\s+#(\d+)";

/// Check if PR body contains a valid closing keyword with issue reference.
pub fn has_closing_keyword(body: &Option<String>) -> bool {
    match body {
        Some(b) if !b.is_empty() => {
            let re = Regex::new(CLOSING_KEYWORDS_PATTERN).expect("Invalid regex pattern");
            re.is_match(b)
        }
        _ => false,
    }
}

/// Extract the issue number from closing keyword, if present.
pub fn extract_closed_issue(body: &Option<String>) -> Option<u64> {
    match body {
        Some(b) if !b.is_empty() => {
            let re = Regex::new(CLOSING_KEYWORDS_PATTERN).expect("Invalid regex pattern");
            re.captures(b).and_then(|caps| caps[3].parse().ok())
        }
        _ => None,
    }
}
```
